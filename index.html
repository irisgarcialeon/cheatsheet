CSS selectors and specifity

CSS Selectors

Selector Description Example
Type selector It selects all elements of the given type within a document. p
Universal selector Selects all elements of any type. *
Class selector Most common ways to select an element. HTML: class="brand", CSS: .brand
Multiple clases It's possible to add more than one class name to an HTML element CSS: .green{color:green} .bold{ font-weight}, HTML: h1 class="green bold"
ID If an HTML needs to be styled uniquely, we can give it an ID using the ID attribute. We prepend the id with (#)
Attribute It can be used to target HTML elements that already contains attributes. [href]{color:magenta}, would target all elements with an href attr.
Pseudo-class Is a keyword added to a selector that specifies a special state of the selected element(s). :hover can be used to change a button's color when the user's pointer hovers over it. button:hover { color: blue;}
